⚙ SETUP-ANLEITUNG FÜR DEN SHOP-BETREIBER

  1. BTCPay Server installieren: Auf btcpayserver.org – kostenlos, open source, selbst gehostet
  2. Lightning Node verbinden: LND oder Core Lightning via BTCPay einrichten
  3. API-Key generieren: In BTCPay Server → Store → Access Tokens
  4. E-Mail SMTP einrichten: Für automatischen STL-Versand (Gmail, Mailgun, etc.)
  5. Webhook konfigurieren: BTCPay → Webhook → deine Server-URL für Auto-Delivery
